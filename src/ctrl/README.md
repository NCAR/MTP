To launch the MTP control program in windows10 on eol-ale:

 * Plug the MTP USB cable into EOL-ale
 * Connect the instrument to the panel
 * Turn on the panel
 * Flip the toggle switch
 * login to eol-ale as lroot
 * Bring up an xterm
 * cd to MTP\src\ctrl

Depending on which platform python3 is installed on:


```

[comment]: #  > python MTPctrl.py

[comment]: #  > python3 MTPctrl.py

python view.py 
python3 view.py

```
